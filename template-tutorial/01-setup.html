1.2 Get tutorial sources

Regarding this workshop, there are two GitHub repositories you will need to fork. Follow the next steps in order to be able to start the workshop.

  • Login in GitHub.

  • Access workshop-gitops-content-deploy repository.

  • Hit Fork available at the top right of the screen. Then click Create fork with the default configuration.

Figure 1. Fork workshop-gitops-content-deploy

Now you have the workshop-gitops-content-deploy repository forked in your GitHub account, let´s clone it in local and create a new branch.

Replace <your_user> for your GitHub user account, and <name> for your assigned cluster in this workshop. 
cd ~
git clone https://github.com/<your_user>/workshop-gitops-content-deploy.git

cd ~/workshop-gitops-content-deploy
git checkout -b sno-<name>-setup

Replicate the same with the second repository.

  • Access helm-infra-gitops-workshop repository.

  • Hit Fork available at the top right of the screen. Then click Create fork with the default configuration.

Figure 2. Fork helm-infra-gitops-workshop

Now you have the helm-infra-gitops-workshop repository forked in your GitHub account, let´s clone it in local and create a new branch.

Replace <your_user> for your GitHub user account, and <name> for your assigned cluster in this workshop. 
cd ~
git clone https://github.com/<your_user>/helm-infra-gitops-workshop.git

cd ~/helm-infra-gitops-workshop
git checkout -b sno-<name>
